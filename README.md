# tampermonkey-scripts
汽水音乐歌词下载油猴脚本

🚀 油猴脚本，支持下载汽水音乐的逐字时间轴LRC歌词

## 功能特性
- 一键下载LRC格式歌词
- 保留逐字时间轴
- 自动提取歌曲元数据

## 安装方法
1. 安装Tampermonkey插件
2. 点击[这里安装脚本](https://greasyfork.org/zh-CN/scripts/537855-%E6%B1%BD%E6%B0%B4%E9%9F%B3%E4%B9%90%E6%AD%8C%E8%AF%8D%E4%B8%8B%E8%BD%BD)
